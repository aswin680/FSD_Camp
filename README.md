# CineVerse – Full Stack Development Camp

A full-stack movie booking platform built progressively over 8 days, covering React frontend, Spring Boot microservices, API Gateway, MongoDB, Redis caching, and RabbitMQ-based async processing.

## Project Structure

| Day | Module | Tech Stack |
|-----|--------|------------|
| Day 02 | React Frontend (Static UI) | React, React Router |
| Day 03 | Auth Service | Spring Boot, JWT, PostgreSQL |
| Day 04 | API Gateway & Microservices | Spring Cloud Gateway, JWT |
| Day 05 | Movie Catalog Service | Spring Boot, MongoDB |
| Day 06 | Booking Service & Seat Management | Spring Boot, PostgreSQL |
| Day 07 | Redis Caching & Performance | Spring Boot, Redis |
| Day 08 | Async Processing (RabbitMQ) | Spring Boot, RabbitMQ |
